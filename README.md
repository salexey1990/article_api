# article_api

### развернуть проект:
- склонировать репозиторий
- установить зависимости `npm i`
- создать базу данных, конфигурационный файл находится по пути `article_api/config/config.json`
- выполнить миграции таблиц в базу данных `node_modules/.bin/sequelize db:migrate`
- запустить проект `node index.js` или `npm run dev` в dev режиме
- открыть в браузере `http://localhost:7000/signup` и зарегистрировать нового пользователя
- пользоваться

### роуты авторизации:
- `http://localhost:7000/signup` - создание нового пользователя
- `http://localhost:7000/signin` - вход под созданным пользователем
- `http://localhost:7000/logout` - выход из сеанса
